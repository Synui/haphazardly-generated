# Haphazardly Generated aka Password Generator

## By Jonathon Schoenholz

## Technologies Used:
* HTML
* CSS
* Javascript (main program)

## Purpose of website:
Utilizes Javascript to randomly generate passwords using combinations of:
* numbers
* letters (uppercase and lowercase)
* special characters
